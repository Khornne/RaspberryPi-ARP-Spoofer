# RaspberryPi-ARP-Spoofer
This repo is my university disertation. The project is creating an RaspberryPi that can attack a network using ARP spoofing. This repository will be contain the code of both the offense script and detection script for ARP spoofing.

### Offense
The offense folder will contain the ARP spoofing script. Inside will have a README containing instructions and explination on how and what is an ARP spoofer.

### Defense
the *Defense* folder will contain the detection script of the ARP spoofer. Inside will contain a README that will contain instructions and explain how detection script works.
